# 悬浮导航栏 - 使用说明

## 功能介绍

悬浮导航栏是一个固定在页面右侧的功能组件，包含以下8个功能模块：

1. **VIP会员** - 展示VIP特权，点击跳转到购买页面
2. **幸运抽奖** - 展示抽奖奖品，点击跳转到抽奖页面
3. **AI智能客服** - 智能客服功能（开发中）
4. **人工客服** - 点击打开客服对话窗口
5. **公众号** - 展示公众号二维码
6. **返回上一页** - 返回浏览器上一页
7. **展开全屏** - 切换浏览器全屏模式
8. **返回顶部** - 平滑滚动到页面顶部

## 安装步骤

### 1. 确保文件已创建

确认以下文件已存在：
- `assets/css/xuwbk_float_nav.css`
- `assets/js/xuwbk_float_nav.js`
- `inc/xuwbk_float_nav_html.php`

### 2. 在主题中添加悬浮导航

编辑主题的 `footer.php` 文件，在 `</body>` 标签之前添加：

```php
<?php get_template_part('inc/xuwbk_float_nav_html'); ?>
```

### 3. 配置后台设置

1. 进入WordPress后台
2. 找到 "XuWbk主题" → "功能组件" → "悬浮导航"
3. 配置以下选项：
   - **启用悬浮导航**: 开启/关闭整个功能
   - **VIP链接**: VIP会员购买页面链接
   - **抽奖链接**: 抽奖活动页面链接
   - **公众号二维码**: 上传二维码图片
   - **客服用户ID**: 客服人员的用户ID

## 配置选项说明

### 基础设置

- **启用悬浮导航**: 总开关，关闭后整个导航栏将不显示
- **电脑端显示**: 控制是否在电脑端显示悬浮导航栏
- **手机端显示**: 控制是否在手机端显示悬浮导航栏

### VIP会员设置

- **VIP链接**: 用户点击VIP图标时跳转的链接地址
- 默认值: `https://bwzy.bwxt88.com/%e4%bc%9a%e5%91%98/`

### 抽奖功能设置

- **抽奖链接**: 用户点击抽奖图标时跳转的链接地址
- 默认值: `https://bwzy.bwxt88.com/%E6%8A%BD%E5%A5%96/`

### 二维码设置

- **公众号二维码**: 上传公众号二维码图片
- 点击二维码可以在新窗口查看大图

### 客服设置

- **客服用户ID**: 客服人员的WordPress用户ID
- 默认值: `1` (管理员)

### 设备显示控制

可以独立控制悬浮导航栏在电脑端和手机端的显示：

- **电脑端显示**: 开启后在电脑端显示，关闭后电脑端不显示
- **手机端显示**: 开启后在手机端显示，关闭后手机端不显示

这样就可以实现：
- 只在电脑端显示，手机端不显示
- 只在手机端显示，电脑端不显示
- 电脑端和手机端都显示
- 电脑端和手机端都不显示

## 功能特点

### 1. 点击跳转功能

- **VIP图标**: 点击图标直接跳转到VIP购买页面
- **抽奖图标**: 点击图标直接跳转到抽奖活动页面
- **后台配置**: 可在后台自由修改跳转链接
- **新窗口打开**: 链接在新窗口打开，不影响当前页面浏览

### 2. 响应式设计

- **桌面端**: 显示在页面右侧
- **平板端**: 缩小尺寸，适应屏幕
- **手机端**: 显示在页面底部，水平排列

### 3. 交互效果

- **鼠标悬停**: 显示工具提示和详情面板
- **平滑滚动**: 返回顶部使用平滑滚动效果
- **动画效果**: 脉冲动画、渐变动画等

### 4. 智能功能

- **客服状态**: 自动检测工作时间（9:00-23:00）
- **返回上一页**: 智能判断是否有上一页，否则返回首页
- **全屏切换**: 支持键盘ESC键退出全屏

### 5. 键盘快捷键

- `Ctrl + ↑`: 返回页面顶部
- `ESC`: 退出全屏模式

## 自定义样式

如果需要自定义样式，可以编辑 `assets/css/xuwbk_float_nav.css` 文件。

主要CSS类：

```css
.comfortable-nav           /* 导航栏容器 */
.nav-item                  /* 导航项 */
.vip-item-right            /* VIP会员 */
.lottery-item             /* 抽奖功能 */
.ai-item                  /* AI客服 */
.service-item             /* 人工客服 */
.qrcode-item              /* 二维码 */
.back-item                /* 返回上一页 */
.fullscreen-item          /* 全屏 */
.backtop-item             /* 返回顶部 */
```

## JavaScript API

悬浮导航提供了全局API，可以在其他脚本中调用：

```javascript
// 显示通知
window.XuWbkFloatNav.showNotification('消息内容', 'success');

// 平滑滚动到指定位置
window.XuWbkFloatNav.smoothScrollTo(0); // 返回顶部

// 检查是否在工作时间
const isWorking = window.XuWbkFloatNav.isWorkingHours();
```

## 常见问题

### 1. 悬浮导航不显示？

- 检查是否在后台启用了该功能
- 确认 `footer.php` 中已添加调用代码
- 检查浏览器控制台是否有错误

### 2. 如何修改颜色？

编辑 `assets/css/xuwbk_float_nav.css`，修改对应的渐变背景：

```css
.vip-item-right {
    background: linear-gradient(135deg, #ffd700 0%, #ff9800 100%);
}
```

### 3. 如何添加新功能？

1. 在HTML中添加新的 `.nav-item`
2. 在CSS中添加对应的样式
3. 在JavaScript中添加交互逻辑

### 4. 客服功能无法使用？

- 确认客服用户ID正确
- 检查主题是否支持私信功能
- 确认用户已登录
